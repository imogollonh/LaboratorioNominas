En el menú de boletos existen 3 opciones disponibles: Tercera edad, Adulto y Niño. En este menú permitirá seleccionar las entradas que se desea adquirir, se tendrán que utilizar los siguientes botones .


Con el signo ( + ) agregamos boletos al contador de compra.




con el botón con signo ( – ) le restamos al contador de boletos.

Una vez determinada la cantidad de boletos requerida le daremos clic al botón siguiente. Para poder continuar tendremos que seleccionar por lo menos un boleto.
